# NodeJS Demos

Demo projects for Laioffer Nodejs class.

##Gulp Demo
1. `$ gulp hello`
2. `$ gulp sass`
3. `$ gulp useref`
4. `$ gulp clean`
5. `$ gulp build`
6. `$ gulp watch`

##Webpack Demo
1. `$ webpack ./entry.js bundle.js`
2. `$ webpack`
3. `$ webpack --progress --colors --watch`
4. `$ webpack-dev-server --progress --colors`
